# 数据库大作业前端

这是一个基于纯 HTML/CSS/JavaScript 的静态图书馆管理系统前端，支持基础管理员流程和演示数据交互。

## 目录结构

- `index.html` - 欢迎/首页，显示当前登录状态与快速入口
- `books.html` - 图书管理，支持增删改查和搜索
- `borrow.html` - 借阅管理，支持查看借阅记录和归还操作
- `readers.html` - 读者管理，支持新增/编辑/删除读者
- `report.html` - 统计报表，显示热门图书和活跃读者
- `admin.html` - 系统安全，支持角色权限、分类管理、借阅规则、备份和恢复
- `script.js` - 前端逻辑与页面交互
- `styles.css` - 全局样式与布局

## 已实现功能

- 登录与会话管理
- 角色选择：系统管理员、图书管理员、读者
- 个人信息编辑与头像上传
- 用户信息持久化到 `localStorage`
- 图书列表搜索、分页、添加、编辑、删除
- 借阅记录查看与归还提示
- 读者信息管理
- 系统安全页面：用户权限、图书分类、借阅规则、备份/恢复演示数据
- 非阻塞提示：`toast` 通知和自定义确认对话框
- 统一 UI 视觉：侧边栏、底部退出登录、紧凑操作按钮、页面布局
- 所有管理页面顶部 welcome 模块已移除，当前用户信息显示在底部退出登录左侧
- 关键按钮已添加稳定 `id` 以便自动化测试和选择器使用

## 使用方法

1. 直接在浏览器中打开 `index.html`。
2. 或者使用本地服务器运行：
   - `python -m http.server 8080`
3. 打开页面后先登录，再访问各个功能页面。

## 说明

- 数据目前基于前端静态数组和 `localStorage`，未对接后端 API。
- 若要对接后端，请在 `script.js` 里替换对应数据读写逻辑。
- `localStorage` 中关键数据包括：
  - `librarySession`：当前登录用户信息
  - `libraryUsers`：用户列表持久化存储

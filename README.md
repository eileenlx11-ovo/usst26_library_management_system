# 借阅系统数据库

图书馆借阅管理系统的数据库设计与实现。

## 项目结构

```
├── README.md                     ← 本文件
├── sql/                          ← 建表语句（按顺序执行）
│   ├── 01_图书模块_建表.sql        ← Author, Publisher, Category, Book
│   ├── 02_借阅模块_建表.sql        ← Reader, Rule, BorrowRecord, Fine, User + 索引
│   └── 03_罚款触发器.sql           ← 触发器（逾期自动罚款 + 库存自动增减）
├── data/                         ← 测试数据
│   ├── 04_全部插入数据.sql         ← 9张表完整测试数据
│   └── 字段查询结果.xls            ← 各表字段结构导出
├── query/                        ← 查询脚本
│   ├── 05_展示查询.sql            ← 答辩演示（统计报表 + 视图 + 存储过程）
│   └── 06_字段查询.sql            ← 查询 INFORMATION_SCHEMA 生成字段报告
└── design/                       ← 设计文档
    ├── 借阅系统数据库设计.md        ← 关系模式 + 范式分析 + 外键关系 + 索引设计
    └── 图书管理系统总E-R图.drawio   ← 完整系统 E-R 图
```

## 执行顺序

```sql
-- 1. 建表（按顺序）
source sql/01_图书模块_建表.sql
source sql/02_借阅模块_建表.sql

-- 2. 插入测试数据
source data/04_全部插入数据.sql

-- 3. 创建触发器
source sql/03_罚款触发器.sql

-- 4. 运行演示查询
source query/05_展示查询.sql
```

## 数据库表概览（9张表）

| 表名 | 说明 | 记录数 | 负责人 |
|------|------|--------|--------|
| Author | 作者信息 | 15 | 成员A |
| Publisher | 出版社信息 | 8 | 成员A |
| Category | 图书分类 | 8 | 成员A |
| Book | 图书信息 | 10 | 成员A |
| Reader | 读者信息 | 20 | 成员B |
| User | 系统用户（登录认证） | 20 | 成员B |
| Rule | 借阅规则 | 4 | 成员B |
| BorrowRecord | 借阅记录 | 30 | 成员B |
| Fine | 罚款记录 | 6 | 成员B |

## 本次更新内容

相比上一版（根目录散落文件），本次主要改动：

**结构调整**
- 重组为 `sql/` `data/` `query/` `design/` 四个目录，按职责分类
- 删除根目录散落的旧文件，统一收入对应目录

**新增内容**
- `User` 表：系统用户表（系统管理员/图书管理员/读者），支持统一登录认证
- 索引设计：7个高频查询索引（借阅记录、读者类型、罚款状态、用户角色）
- `BorrowRecord.fk_borrow_book` 外键：补全对 Book 表的外键约束
- 库存触发器：`trg_borrow_decrease_stock`（借出扣库存）、`trg_return_increase_stock`（归还恢复库存）
- 展示查询扩展：视图演示（5个视图）、存储过程演示（借/还/续/缴罚款）、错误处理演示
- 设计文档：`design/借阅系统数据库设计.md`（关系模式 + 范式分析 + 外键关系 + 索引说明）

**数据不变**
- 原有8张表的建表语句和插入数据与上一版完全一致，无任何改动
